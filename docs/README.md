## For Beginners

1. Start with **[01 - Getting Started](01-getting-started.md)** to install and run the bot in **dry run** (no real orders).
2. Read **[02 - Configuration](02-configuration.md)** to set credentials and optional parameters.
3. Read **[03 - Trading Strategy](03-trading-strategy.md)** to understand when the bot trades and how risk is controlled.

Only after you’re comfortable with dry-run behavior and settings should you consider running with `DRY_RUN=0` for live trading.
